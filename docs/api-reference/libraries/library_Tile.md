The library that provides functions for manipulating tiles in blur's default `World`

| **Tile** new( **Vector** position, **Texture** texture )  |
| ------------------- |
| Creates a new Tile |



| remove( **Tile** tile )  |
| ------------------- |
| Removes from the world and disposes of a Tile object |
